# Генератор анатомических исследований эпохи Возрождения

Этот проект создает изображения в стиле академических анатомических исследований эпохи Возрождения, вдохновленные работами Леонардо да Винчи и других мастеров того времени.

## Особенности

- **Анатомическая фигура**: Детализированная фигура человека с основными органами
- **Стиль эпохи Возрождения**: Латинские надписи, декоративные элементы
- **Эффекты старения**: Текстура старой бумаги, пятна, складки
- **Академический подход**: Измерительные линии, научные обозначения

## Установка

1. Убедитесь, что у вас установлен Python 3.7+
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Запустите генератор:

```bash
python renaissance_anatomy_generator.py
```

Изображение будет сохранено как `renaissance_anatomy_study.png` в текущей директории.

## Структура изображения

### Основные элементы:
- **Голова**: Круглая форма с анатомическими деталями
- **Туловище**: Прямоугольная форма с внутренними органами
- **Конечности**: Руки и ноги в анатомическом стиле
- **Внутренние органы**:
  - Сердце (COR)
  - Легкие (PULM)
  - Печень (HEPAR)
  - Кишечник

### Декоративные элементы:
- Латинские надписи
- Декоративная рамка
- Измерительные линии
- Эффекты старения бумаги

## Настройка

Вы можете изменить параметры генератора, отредактировав класс `RenaissanceAnatomyGenerator`:

- Размер изображения: измените `width` и `height` в конструкторе
- Цвета: настройте цветовые коды в методах рисования
- Детали: добавьте или измените анатомические элементы

## Примеры использования

```python
# Создание генератора с пользовательскими размерами
generator = RenaissanceAnatomyGenerator(800, 1000)

# Генерация изображения
image = generator.generate()

# Сохранение с пользовательским именем
generator.save("my_anatomy_study.png")
```

## Исторический контекст

Этот генератор вдохновлен работами:
- Леонардо да Винчи (1452-1519)
- Андреаса Везалия (1514-1564)
- Других анатомов эпохи Возрождения

Изображения создаются в стиле, характерном для научных исследований XV-XVI веков, когда анатомия стала важной областью научного изучения.
