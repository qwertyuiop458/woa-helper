# 🎨 Renaissance Anatomy Generator - Полный проект

Создание изображений в стиле академических анатомических исследований эпохи Возрождения с современным веб-интерфейсом.

## 📋 Содержание проекта

### 🖼️ Генераторы изображений
- **Базовый генератор** - классическая анатомическая фигура
- **Продвинутый генератор** - детализированная анатомия с кровеносной системой

### 🌐 Веб-интерфейс
- **Современный дизайн** - адаптивный интерфейс
- **Интерактивность** - анимации и эффекты
- **Галерея работ** - демонстрация результатов

## 🚀 Быстрый старт

### Вариант 1: Веб-интерфейс (Рекомендуется)

# Запуск веб-сервера

## 🎨 Особенности

### Генераторы изображений
- **Анатомическая точность** - детализированные органы
- **Стиль эпохи Возрождения** - латинские надписи, декоративные элементы
- **Эффекты старения** - текстура старой бумаги, пятна, складки
- **Научные обозначения** - COR, PULMONES, HEPAR, VENTRICULUS, PANCREAS

- **Интерактивность** - выбор типа генератора, анимации загрузки
- **Галерея работ** - демонстрация созданных изображений
- **Функции скачивания** - сохранение результатов

## 🎯 Использование

### Через веб-интерфейс
1. Запустите `python3 server.py`
2. Откройте браузер: http://localhost:8080
3. Выберите тип генератора
4. Нажмите "Создать изображение"
5. Скачайте результат

### Через командную строку
```bash
# Установка зависимостей
pip install --break-system-packages -r requirements.txt

# Запуск генераторов
python3 renaissance_anatomy_generator.py
python3 advanced_renaissance_generator.py
```
