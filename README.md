# Zombie Infection - Java Game

## Описание игры

Zombie Infection - это 2D игра-шутер, созданная на Java с использованием Swing. Игрок должен выживать, сражаясь с волнами зомби.

## Особенности игры

- **Управление персонажем**: WASD или стрелки для движения
- **Стрельба**: Пробел для стрельбы
- **Волны зомби**: Каждая волна содержит больше зомби
- **Система здоровья**: Игрок теряет здоровье при контакте с зомби
- **Боеприпасы**: Ограниченное количество патронов
- **Бонусы**: Случайные бонусы здоровья и патронов
- **Система очков**: Очки за уничтожение зомби

## Управление

- **W/↑**: Движение вверх
- **S/↓**: Движение вниз
- **A/←**: Движение влево
- **D/→**: Движение вправо
- **Пробел**: Стрельба
- **R**: Перезапуск игры (после проигрыша или победы)

## Цель игры

Выжить 10 волн зомби. Каждая волна становится сложнее, так как количество зомби увеличивается.

## Компиляция и запуск

```bash
# Компиляция
javac *.java

# Запуск
java ZombieInfection
```

## Требования

- Java 8 или выше
- Графическая среда (для отображения окна)

## Структура проекта

- `ZombieInfection.java` - Основной класс игры
- `Player.java` - Класс игрока
- `Zombie.java` - Класс зомби
- `Bullet.java` - Класс пуль
- `PowerUp.java` - Класс бонусов

## Геймплей

1. Управляйте синим персонажем
2. Избегайте зеленых зомби
3. Стреляйте в зомби желтыми пулями
4. Собирайте красные (здоровье) и желтые (патроны) бонусы
5. Выживите 10 волн для победы

Удачи в игре!
