# Генератор анатомических исследований эпохи Возрождения

Этот проект создает изображения в стиле академических анатомических исследований эпохи Возрождения, вдохновленные работами Леонардо да Винчи, Андреаса Везалия и других мастеров того времени.

## 🎨 Особенности

### Базовый генератор (`renaissance_anatomy_generator.py`)
- **Анатомическая фигура**: Базовая фигура человека с основными органами
- **Стиль эпохи Возрождения**: Латинские надписи, декоративные элементы
- **Эффекты старения**: Текстура старой бумаги, пятна, складки
- **Академический подход**: Измерительные линии, научные обозначения

### Продвинутый генератор (`advanced_renaissance_generator.py`)
- **Детализированная анатомия**: Более сложная фигура с деталями
- **Внутренние органы**: Сердце, легкие, печень, желудок, поджелудочная железа
- **Кровеносная система**: Аорта, вены, капилляры
- **Реалистичная текстура**: Улучшенная имитация старой бумаги
- **Научные обозначения**: Латинские термины для всех органов

## 📋 Требования

- Python 3.7+
- Pillow >= 9.0.0
- matplotlib >= 3.5.0
- numpy >= 1.21.0

## 🚀 Установка и использование

### 1. Установка зависимостей
```bash
pip install --break-system-packages -r requirements.txt
```

### 2. Запуск базового генератора
```bash
python3 renaissance_anatomy_generator.py
```

### 3. Запуск продвинутого генератора
```bash
python3 advanced_renaissance_generator.py
```

## 🎯 Структура изображения

### Основные элементы:
- **Голова**: Круглая форма с анатомическими деталями
- **Туловище**: Прямоугольная форма с внутренними органами
- **Конечности**: Руки и ноги в анатомическом стиле
- **Внутренние органы**:
  - Сердце (COR)
  - Легкие (PULMONES)
  - Печень (HEPAR)
  - Желудок (VENTRICULUS)
  - Поджелудочная железа (PANCREAS)
  - Кишечник

### Декоративные элементы:
- Латинские надписи
- Декоративная рамка с угловыми украшениями
- Измерительные линии с числовыми отметками
- Индикатор масштаба
- Эффекты старения бумаги

## 📚 Исторический контекст

Этот генератор вдохновлен работами:
- **Леонардо да Винчи** (1452-1519) - Анатомические зарисовки
- **Андреаса Везалия** (1514-1564) - "De humani corporis fabrica"
- **Других анатомов эпохи Возрождения**

Изображения создаются в стиле, характерном для научных исследований XV-XVI веков, когда анатомия стала важной областью научного изучения.
