# Прогноз погоди - Повна копія сайту

Це повна копія сайту прогнозу погоди з усіма функціями, створена з використанням сучасних веб-технологій.

## 🌟 Особливості

### 📱 Головна сторінка (index.html)
- **Поточні погодні умови** з детальною інформацією
- **Розширений прогноз на 10 днів** з можливістю перемикання періодів
- **Погодинний прогноз** з горизонтальною прокруткою
- **Карти погоди** (заглушки для інтеграції)
- **Адаптивний дизайн** для всіх пристроїв

### 🌡️ Сторінка поточних умов (current-weather.html)
- **Детальна інформація про погоду** (температура, вітер, вологість, тиск)
- **Якість повітря** з індексом AQI та деталями забруднювачів
- **Попередження про погоду** з пріоритетами
- **Сонячна активність** (схід/захід сонця, тривалість дня)

### 🗺️ Карти погоди (weather-maps.html)
- **Інтерактивна карта** з використанням Leaflet.js
- **Різні типи карт**: температура, опади, вітер, тиск, вологість, хмарність
- **Метеостанції** з реальними даними
- **Супутникові зображення** (заглушки)
- **Радар опадів** з часовими зсувами

### 📰 Новини погоди (news.html)
- **Головна новина** з великим зображенням
- **Категорії новин**: прогнози, клімат, стихійні лиха, наука
- **Пошук новин** з фільтрацією
- **Модальні вікна** для детального перегляду
- **Кліматичні дані** за останній місяць

## 🛠️ Технології

- **HTML5** - семантична розмітка
- **CSS3** - сучасні стилі з Flexbox та Grid
- **JavaScript (ES6+)** - інтерактивність та логіка
- **Leaflet.js** - інтерактивні карти
- **Font Awesome** - іконки
- **Google Fonts** - типографіка

## 🚀 Швидкий старт

1. **Клонуйте репозиторій**:
   ```bash
   git clone <repository-url>
   cd weather-forecast-site
   ```

2. **Відкрийте головну сторінку**:
   ```bash
   # Просто відкрийте index.html у браузері
   # Або використайте локальний сервер
   python -m http.server 8000
   # Потім перейдіть на http://localhost:8000
   ```

3. **Налаштування API** (опціонально):
   - Отримайте API ключ від OpenWeatherMap
   - Замініть `YOUR_API_KEY` у файлах `script.js`, `current-weather.js`, `weather-maps.js`
   - Розкоментуйте код для реальних API викликів

## 🔧 Налаштування

### Інтеграція з реальними API

1. **OpenWeatherMap API**:
   ```javascript
   // У файлах script.js, current-weather.js, weather-maps.js
   this.apiKey = 'YOUR_ACTUAL_API_KEY';
   
   // Розкоментуйте методи fetchWeatherData та fetchForecastData
   ```

2. **Air Quality API**:
   ```javascript
   // Додайте інтеграцію з API якості повітря
   async fetchAirQualityData(city) {
       const response = await fetch(
           `https://api.openweathermap.org/data/2.5/air_pollution?q=${city}&appid=${this.apiKey}`
       );
       return await response.json();
   }
   ```

3. **Weather Alerts API**:
   ```javascript
   // Додайте інтеграцію з API попереджень
   async fetchWeatherAlerts(city) {
       const response = await fetch(
           `https://api.openweathermap.org/data/2.5/onecall?q=${city}&exclude=current,minutely,hourly,daily&appid=${this.apiKey}`
       );
       return await response.json();
   }
   ```

### Налаштування карт

1. **Додайте реальні дані погоди**:
   ```javascript
   // У weather-maps.js замініть generateMockWeatherData
   async generateRealWeatherData() {
       const response = await fetch(
           `${this.baseUrl}/forecast?q=Ukraine&appid=${this.apiKey}&units=metric`
       );
       return await response.json();
   }
   ```

2. **Інтеграція з супутниковими даними**:
   ```javascript
   // Додайте реальні супутникові зображення
   updateSatelliteImage(type) {
       const satelliteImage = document.getElementById('satelliteImage');
       satelliteImage.innerHTML = `
           <img src="https://api.satellite.com/${type}/ukraine.png" alt="Satellite">
       `;
   }
   ```

## 🎯 Функціональність

### Пошук міст
- Введіть назву міста в пошукове поле
- Натисніть Enter або кнопку пошуку
- Отримайте прогноз погоди для обраного міста

### Фільтрація новин
- Клікніть на категорію новин
- Переглядайте новини за категоріями
- Використовуйте пошук для знаходження конкретних новин

### Інтерактивні карти
- Перемикайте типи карт (температура, опади, вітер тощо)
- Клікайте на карту для отримання інформації
- Переглядайте легенду для розуміння кольорів

### Погодинний прогноз
- Прокручуйте горизонтально для перегляду всіх годин
- Перемикайте періоди прогнозу (3, 7, 10 днів)
