# 🌐 Веб-страница генератора анатомических исследований эпохи Возрождения

Современная веб-страница для демонстрации и использования генератора анатомических изображений в стиле эпохи Возрождения.

## 🚀 Быстрый старт

### 1. Запуск веб-сервера
```bash
python3 server.py
```

### 2. Открытие в браузере
Автоматически откроется браузер с адресом:
- http://localhost:8080
- или http://127.0.0.1:8080

## 📁 Структура веб-проекта

```
├── index.html                    # Главная HTML страница
├── styles.css                    # CSS стили
├── script.js                     # JavaScript функциональность
├── server.py                     # Веб-сервер
├── renaissance_anatomy_study.png # Базовое изображение
├── advanced_renaissance_anatomy.png # Продвинутое изображение
└── README_web.md                # Эта документация
```

## 🔧 Настройка и кастомизация

### Изменение порта
Отредактируйте файл `server.py`:
```python
port = 8080  # Измените на нужный порт
```

### Добавление новых изображений
1. Добавьте PNG файлы в директорию
2. Обновите галерею в `index.html`
3. Добавьте ссылки в `script.js`

### Изменение стилей
Отредактируйте `styles.css`:
```css
/* Изменение цветовой схемы */
:root {
    --primary-color: #8B0000;
    --accent-color: #DC143C;
}
```
